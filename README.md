# CharacterSheet

A D&D 5e character sheet project using **React + Vite + TypeScript** with Tailwind CSS.

## Current app state

The app now includes an initial architecture skeleton:

- typed character sheet model (`src/model/characterSheet.ts`)
- editable canvas shell in `src/App.tsx`
- working **Add section** proof flow in edit mode

## Scripts

```bash
npm install
npm run dev
npm run lint
npm run build
```
